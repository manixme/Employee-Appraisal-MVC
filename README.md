# 👨‍💼 Employee Performance Appraisal System

A full-stack **ASP.NET Core MVC application** to manage employee records and performance efficiently.

---

## 🚀 Features

- ➕ Add Employees
- 📋 View Employee List
- ✏️ Edit Employee Details
- ❌ Delete Employees
- ⭐ Rating System (1–5 dropdown)
- ✅ Active / Inactive Status Display
- 🎨 Styled UI using Bootstrap

---

## 🧱 Tech Stack

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap

---

## 🧠 What I Learned

- MVC Architecture (Model, View, Controller)
- CRUD Operations with EF Core
- Routing & Tag Helpers
- Model Binding
- Razor Views
- Debugging real-world errors
